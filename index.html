Participation Guidelines

Rules

  • You have 90 minutes to build or sketch.
  • Work solo or in a pair.
  • All work should be created during the hackathon, but reusing your own snippets is fine.
  • AI tools are allowed‚Äîjust be ready to explain your process.

Submission Format

  • Create a GitHub repository for your project.
  • Include your code + any other materials in the repo.
  • Post your project in the GH repo Issue thread for the relevant challenge. Include title, hacker(s), description, repo url and a screenshot.
